Post Install script for Debian based Linux Distros
Available on my GitLab page Here!

This was my end of year project for my computing diploma at South Devon College. The script was designed to be run after installing your desired Debian based OS. The main functionality of the script was to automate the configuration process. The project was written in BASH and is around 250 lines. (not including the config files)

Features:
  • Install and setup firewall
  • Install a user created list of desired application
  • Setup Snapd
  • Install any user specified snap packages
  • Setup Flatpak
  • Install any user specified flatpak apps
  • Move the users wallpapers into the /usr/share/backgrounds thus allowing them to be used by all users
  • Move the users scripts to /usr/bin/ to make them executable from everywhere
  • Swap some config files for user created ones
  • Change the users shell to FISH which is brilliant!
  • Remove some bloatware which comes with distros such as Ubuntu
