Key



This web app has to ability to control the lights in SMB109. The lights are by default set to red. The application will set specific lights to green depending on the features present in the password. Bellow is the key.




No secure features present

When there are no secure password features present in the password all lights will show red.

Length over 8 characters

When the password length is over 8 characters, light 1 will illuminated in green.

Length over 16 characters

When the password length is over 16 characters, light 2 will illuminated in green.

Multicase

When the password contains both upper and lower case characters

Special charecters

When the password contains special charecters (*,&,Â£,",., etc... ) light 4 will illuminate in green.

Alphanumeric

When the password is alphanumeric (contains both letters and numbers) light 5 will illuminate in green.

No repeating Characters

When there are no instances where three of the same character repeat themselves light 6 wil illuminate in green.
