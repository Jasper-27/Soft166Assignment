About the application



This is a password strength evaluation web app. When the password is entered into the input field on the application page, the site will automatically evaluate the strength of the password based on 5 characteristics:

  • Length
  • Presence of Special Characters
  • Presence of Alphanumeric
  • Presence of Multi-case
  • Presence of Consecutive characters

Once when the application detects one of these features it will make the description of the feature as well as the corresponding light on the wall, change from red to green. If that feature is removed then the light and text will revert to red. This all happens in real time, no need to submit the password.
